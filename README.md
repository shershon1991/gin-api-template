# gin-import-template
`Gin`框架学习使用,并实践常用包在Gin框架中的集成和使用。[github地址: https://github.com/shershon1991/gin-api-template](https://github.com/shershon1991/gin-api-template.git)

# 目录介绍
```
├── api // 接口
├── config // 配置
├── core // 核心代码
├── cron // 定时任务
├── global // 全局变量和常量
├── initialize // 初始化相关
├── logs // 日志目录
├── middleware // 中间件
├── main.go // 启动文件
├── model 
    └── dao // 操作数据库方法
    └── entity // 和数据库表结构一一对应
    └── request // 请求参数结构体
    └── response // 返回参数结构、返回函数封装
├── router // 路由
├── test // 单元测试目录
└── utils // 工具包
```
